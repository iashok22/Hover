How to use

1. Add This HTML

<div class="row grid cs-style-1"> <div class="col-md-6 col-sm-6"> <figure> <img class="img-responsive" src="images/Bob.jpg" alt="img01"> <figcaption> <h3 class="text-center">Bob Dylan</h3> <p class="text-center">Once upon a time</p> <a href="#">Rolling Stone</a> </figcaption> </figure> </div> <div class="col-md-6 col-sm-6"> <figure> <img class="img-responsive" src="images/Bob.jpg" alt="img01"> <figcaption> <h3 class="text-center">Bob Dylan</h3> <p class="text-center">Once upon a time</p> <a href="#">Rolling Stone</a> </figcaption> </figure> </div> </div>

2. Choose Hover Style from below e.g. Style 7

3. Add CSS from that style

.cs-style-7{ figure img { z-index: 12; } figcaption { height: 100%; width: 100%; opacity: 0; position: absolute; top: 0; left: 0; padding: 5px; background: #333; color: $primary-color; -webkit-backface-visibility: hidden; -moz-backface-visibility: hidden; backface-visibility: hidden; -webkit-transition: opacity 0.3s, height 0.3s, box-shadow 0.3s; -moz-transition: opacity 0.3s, height 0.3s, box-shadow 0.3s; transition: opacity 0.3s, height 0.3s, box-shadow 0.3s; box-shadow: 0 0 0 0px #2c3f52; z-index: 11; } figure:hover figcaption, figure.cs-hover figcaption { opacity: 1; height: 130%; box-shadow: 0 0 0 10px $primary-color; } figcaption h3 { margin-top: 68%; } figcaption h3, figcaption span, figcaption a { opacity: 0; -webkit-transition: opacity 0s; -moz-transition: opacity 0s; transition: opacity 0s; } figure:hover figcaption h3, figure:hover figcaption span, figure:hover figcaption a, figure.cs-hover figcaption h3, figure.cs-hover figcaption span, figure.cs-hover figcaption a { -webkit-transition: opacity 0.3s 0.2s; -moz-transition: opacity 0.3s 0.2s; transition: opacity 0.3s 0.2s; opacity: 1; } figcaption a { text-align: center; padding: 9px 24px; border-radius: 2px; display:block; background: $primary-color; color: #fff; text-decoration: none; width: 40%; margin: auto; } }

4. Change cs-style-1 in HTML line 1 to match your style e.g cs-style-7

5. Done!

For Icon Hover (cs-style-9 change :

<figcaption> <h3 class="text-center">Bob Dylan</h3> <p class="text-center">Once upon a time</p> <a href="#">Rolling Stone</a> </figcaption>

To this :

<figcaption> <i class="fa fa-3x fa-search"></i> </figcaption>
